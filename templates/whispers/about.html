{% trans "About psst-secret" %}

{% trans "A privacy focused app for sharing secrets securely. We call them whispers." %}

{% trans "How It Works" %}

{% trans 'psst-secret ensures that your whispers never exist in plaintext on our server. All encryption and decryption happens entirely in your browser using the Web Crypto API.' %}

🔒 {% trans "Creating a Whisper" %}

  1. {% trans "You type your whisper in the browser" %}
  2. {% trans 'A random AES-256-GCM key is generated client-side' %}
  3. {% trans "Your whisper is encrypted in-browser" %}
  4. {% trans "Only the ciphertext is sent to the server" %}
  5. {% trans 'The decryption key is placed in the URL fragment (#key)' %}

🔓 {% trans "Viewing a Whisper" %}

  1. {% trans "You open the shared link" %}
  2. {% trans "The server returns the ciphertext" %}
  3. {% trans "The key from the URL fragment decrypts it in-browser" %}
  4. {% trans "You see the plaintext — the server never does" %}

{% trans "Why the URL Fragment?" %}

{% trans "The decryption key is stored after the # in the URL. Per the HTTP specification, the URL fragment is never sent to the server — it's processed only by the browser. This means:" %}

  • {% trans "The server never receives the decryption key" %}
  • {% trans "The key doesn't appear in server logs or analytics" %}
  • {% trans "Even if the server is compromised, your whispers remain encrypted" %}

🔑 {% trans "Password Protection" %}

{% trans "Optionally, you can set a password when creating a whisper. When a password is used:" %}

  • {% trans 'The encryption key is derived from your password using PBKDF2 (600,000 iterations, SHA-256)' %}
  • {% trans 'A random salt is generated and stored on the server (useless without the password)' %}
  • {% trans "The URL contains no key fragment — only the password holder can decrypt" %}
  • {% trans "This provides an extra layer of security for highly sensitive data" %}

{% trans "Security Features" %}

🔥 {% trans "Burn After Reading" %}

{% trans 'Whispers marked as burn-after-read are permanently erased from memory the moment they are viewed — not just flagged, but gone.' %}

⏰ {% trans "Auto-Expiry" %}

{% trans "All whispers require an expiration time (5 minutes to 1 month). Expired whispers are automatically evicted from Redis via TTL and continuously purged by a background process." %}

🛡️ AES-256-GCM

{% trans "Industry-standard authenticated encryption. Provides both confidentiality and integrity — any tampering with the ciphertext is detected." %}

🚫 {% trans "Zero Knowledge" %}

{% trans "The server stores only ciphertext, IV, and optional salt — and only in volatile memory (Redis). It has no ability to decrypt your data, even under compulsion." %}
